# Task Manager

A beautiful, responsive, and modern Task Manager web application built with HTML, CSS, and Vanilla JavaScript.

## ✨ Features

- **Add Tasks**: Quickly add new tasks to your to-do list.
- **Mark as Complete**: Toggle tasks between active and completed states.
- **Delete Tasks**: Remove tasks you no longer need.
- **Local Storage**: Your tasks are automatically saved to your browser's local storage so you never lose them when you refresh or close the page.
- **Modern UI**: Features a stunning, bright glassmorphism design with animated background shapes and smooth transition animations.
- **Responsive**: Works perfectly on both desktop and mobile devices.
- **Empty State**: Friendly visual indicator when you have no pending tasks.

## 🚀 Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Custom properties (variables), Flexbox, Animations, Glassmorphism (backdrop-filter), gradients.
- **JavaScript (ES6+)**: DOM manipulation, event listeners, LocalStorage API.
- **Ionicons**: Modern, scalable vector icons.
- **Google Fonts**: Uses the 'Inter' typeface for clean, highly legible text.

## 🛠️ Setup and Usage

1. Clone or download this repository.
2. No build tools or package managers are required!
3. Simply open `index.html` in any modern web browser to start using the app.

## 📁 File Structure

- `index.html` - The main structure of the application.
- `style.css` - All styling, theme variables, and animations.
- `script.js` - Application logic, state management, and local storage handling.

## 🎨 Design

The app uses a bright and airy theme (`#f0fdfa`) accented with vibrant cyan (`#06b6d4`), rose (`#f43f5e`), and purple (`#8b5cf6`) elements to create a fresh, modern aesthetic. The main task container utilizes a translucent glass effect to blend beautifully with the animated background shapes.
